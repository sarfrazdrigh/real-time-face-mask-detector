# Real-Time Face Mask Detector

An Android application that detects face masks in real-time using TensorFlow Lite.

## Features
- Real-time face mask detection.
- Front camera support.
- Simple and user-friendly UI.

## Technologies Used
- Android Studio
- Java
- TensorFlow Lite
